# Como baixar ISO Windows 10
 
* O que é imagem ISO?
É um arquivo que contém uma **cópia exata** de um sistema de arquivos de um disco óptico(CD/DVD).<br>
Permite instalar o SO (sistema operacional) em uma máquina física ou virtual.
